# POS using rails
  An open source POS using rails. Hopefully, it can support real world scenario.

# Specs
* Ruby ``` 2.4.2 ```
* Rails ```5.1.4```

# Setting Up
## Database
  This app uses ```postgresql```

  Run ```rake db:create db:migrate db:seed```

## Running Test
```bundle exec rspec```
